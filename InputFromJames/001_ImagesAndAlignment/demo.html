How to align text ( left, right & centre ) in HTML

Some useful reading:

  • What is HTML?
  • Creating a paragraph in HTML
  • What is CSS?
  • Aligning text with CSS
  • What is a class in HTML?

This text is centred because this <p> tag has a class of "center", and the CSS stylesheet applies the rule text-align: center; to paragraphs with this class.

This text is left-aligned because the CSS `text-align` property of the <body> of the page is set to `left`, and this <p> element inherits its style from the <body> element.

This text is right-aligned using a 'style' attribute attached directly to its <p> tag.
